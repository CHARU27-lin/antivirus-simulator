# 🛡️ Antivirus Scanner

A Python-based antivirus simulator built from scratch with
both terminal and GUI interfaces.

## ✨ Features

- 🔍 Scan single files
- 📁 Scan entire folders
- 🦠 Virus signature detection
- ⏱️ Real time file monitoring
- 🚫 Auto quarantine infected files
- 📄 Scan report generation
- 🖥️ GUI interface

## 🛠️ Built With

- Python 3.x
- Tkinter (GUI)
- Watchdog (Real time monitor)
- Hashlib (File hashing)
